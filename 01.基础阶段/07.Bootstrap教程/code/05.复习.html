1.less继承(类继承)
              #test {
                  &:extend(.father)
              }
              // 不推荐
              #test:extend(.father) {}
              继承实质是将.father和#test组合成一个选择器，声明快使用.father的
              all: 继承所有和.father相关的声明块
              父类不能定义成混合(继承的灵活性低，性能高)
            2.less的避免编译
              ~"不会被编译的内容"
              变量在less中属于块级作用域，延迟加载
